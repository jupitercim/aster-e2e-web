# AsterDEX 1001倍高杠杆交易

## Application Overview

AsterDEX 1001x 高杠杆交易页面（/zh-CN/trade/1001x/futures/BTCUSD）的功能测试，验证页面加载（杠杆信息/做多做空按钮）、永续合约与预测模式 Tab 切换、底部仓位/订单/历史 Tab 切换。测试使用已登录的 MetaMask 钱包 fixture（loggedInPage）。

## Test Scenarios

### 1. AsterDEX - 1001倍高杠杆交易

**Seed:** `tests/cases/seed.spec.ts`

#### 1.1. 1001x 页面正常加载，显示杠杆滑动条和做多/做空按钮

**File:** `tests/cases/1001x.spec.ts`

**Steps:**
  1. 导航至 1001x URL（/zh-CN/trade/1001x/futures/BTCUSD），等待 domcontentloaded 后再等待 3 秒
     - expect: 页面标题不为空（soft 断言）
  2. 查找做多按钮（label:has-text("做多") 等）
     - expect: 做多按钮状态记录（label 可能被容器裁剪，不作强断言）
  3. 查找做空按钮
     - expect: 做空按钮状态记录
  4. 查找杠杆信息（1001/Degen/杠杆/Leverage 等关键词）
     - expect: 找到至少一个杠杆元素（soft 断言）

#### 1.2. 永续合约与预测模式 Tab 切换正常

**File:** `tests/cases/1001x.spec.ts`

**Steps:**
  1. 点击「预测」Tab（若可见），等待 1.5 秒后查找预测 UI 元素
     - expect: 预测 UI 元素（预测/上涨/下跌/Up/Down 等）可见；若 Tab 不存在则跳过
  2. 切换回「永续合约」Tab
     - expect: 永续合约模式恢复

#### 1.3. 仓位、订单、历史 Tab 切换正常

**File:** `tests/cases/1001x.spec.ts`

**Steps:**
  1. 点击底部仓位/订单/历史 Tab 之一（遍历多种文案）
     - expect: 找到可见 Tab 并点击
  2. 验证内容区域存在（暂无资料/No Data/仓位/订单 等关键词）
     - expect: 内容区域有内容（有数据或空状态提示）
